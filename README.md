## About

I'm building a complete, publicly documented portfolio in GIS, remote sensing, and drone mapping — combining rigorous technical analysis with professional cartographic outputs.

My work focuses on vegetation monitoring, land use classification, photogrammetry, thermal inspection, and Python-based geospatial automation. Each project integrates real satellite and drone data, follows reproducible workflows, and delivers professional-grade deliverables: PDF maps at 300dpi, interactive web maps, technical reports, and statistical datasets.

**Target sectors:** forestry, precision agriculture, agroindustrial monitoring, oil & gas, infrastructure inspection.
